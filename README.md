# Website

Auto-generiert von **Thunder v4** (JSON → Astro Pipeline).

## Lokal starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Komponenten und Styles stammen aus der Thunder-Bibliothek (`@thunder/library`).
Brand-Tokens (Farben, Schriften, Radius) sind in `src/styles/tokens.css`
festgehalten und überschreiben die Default-Variablen aus der Bibliothek.
